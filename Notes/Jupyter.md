## Create Cells
- Above: `Esc` + `A`
- Below: `Esc` + `B`

## Change Cells
- To Markdown: `Esc` + `M`
- To Code: `Esc` + `Y`

## Delete / Cut / Copy / Paste Cells
- Delete: `Esc` + `D D`
- Cut: `Esc` + `X`
- Copy: `Esc` + `C`
- Paste: `Esc` + `V`

## Run / Save Cells
- Run: `Shift` + `Enter`
- Run and insert below: `Alt`+`Enter`
- Save: `Ctrl`+`S`

## Change Mode
- Command: `Esc`
- Edit: `Enter`

## Other
- Undo: `Esc`+`Z`
- Select Multiple: `Shift`+`Up/Down`
- Move cell up/down: `Esc` + `K` or `J`
